# BrainGPS — навигатор мышления в сложных ситуациях

> «GPS для мышления»: помогает быстро понять, что за ситуация происходит,
> какие типовые ошибки в ней совершают и какое поведение здесь рационально.

BrainGPS — это Telegram-бот, который помогает сориентироваться в сложной
социальной или переговорной ситуации и принять решение в моменте.
Это **не терапия, не психология и не «разговоры о чувствах»** — это инструмент
когнитивной навигации.

## Какую проблему решает

В конфликте, под давлением или в манипуляции человек реагирует импульсивно,
тонет в эмоциях и совершает типовые ошибки. BrainGPS не успокаивает и не лечит —
он помогает **увидеть структуру ситуации** и действовать осознанно.

## Как это работает

1. **Классификация.** Пользователь не «рассказывает душу», а отвечает на короткие
   вопросы. По ответам бот определяет тип ситуации (конфликт, давление,
   манипуляция, ответственность, неопределённость, напряжение в отношениях и т.д.).
2. **Фиксация рамок.** Система жёстко задаёт контекст: цель, ограничения, риски и
   когнитивные ловушки, характерные именно для этого типа ситуации.
3. **Генерация внутри рамок.** Только теперь подключается LLM — и выдаёт не
   абстрактный совет, а конкретные варианты действий, формулировки фраз и стратегию
   поведения строго под данный тип ситуации.
4. **Тренажёр (симуляция).** Отдельный режим, где бот играет оппонента по диалогу,
   чтобы отработать поведение до реального разговора.

Ключевая идея: масштабируется не за счёт опыта конкретного консультанта, а за счёт
**формализации повторяющихся человеческих ситуаций и паттернов поведения**.

## Архитектура

```
Пользователь → Telegram Bot
                  │
                  ▼
        Классификатор ситуации        (короткий опрос → тип ситуации)
                  │
                  ▼
        Контекст-движок               (цель, ограничения, риски, ловушки)
                  │
                  ▼
            LLM-генерация             (действия/фразы/стратегия внутри рамок)
                  │
                  ├──► Ответ пользователю
                  └──► Режим-тренажёр  (симуляция оппонента)
```

- **Язык:** Python
- **Интерфейс:** Telegram Bot API
- **LLM:** Groq API (`<модель, напр. llama-3.3-70b>`)
- **Запуск:** Docker + docker-compose

## Быстрый старт

```bash
git clone https://github.com/Keysiks/BrainGps.git
cd BrainGps

cp .env.example .env
# заполните переменные окружения (см. ниже)

docker compose up --build
```

### Переменные окружения

| Переменная        | Описание                          |
|-------------------|-----------------------------------|
| `TELEGRAM_TOKEN`  | токен бота от @BotFather          |
| `GROQ_API_KEY`    | ключ доступа к Groq API           |
