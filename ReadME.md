Это система когнитивной навигации для человека в сложных социальных и жизненных ситуациях. Не терапия, не психология и не «разговоры о чувствах», а инструмент, который помогает быстро понять, **что за ситуация происходит, какие типовые ошибки в ней совершают люди и какое поведение здесь рационально**.

В основе лежит дерево (или граф) жизненных ситуаций: конфликт, давление, манипуляция, ответственность, неопределённость, напряжение в отношениях и т.д. Пользователь не «рассказывает душу», а отвечает на короткие вопросы, которые позволяют классифицировать происходящее. После классификации система жёстко фиксирует рамки контекста: цель, ограничения, риски, когнитивные ловушки.

И только внутри этих рамок подключается LLM, которая генерирует варианты действий, формулировки фраз или стратегию поведения — не абстрактно, а строго под данный тип ситуации. За счёт этого человек перестаёт импульсивно реагировать, не утопает в эмоциях и не совершает типовые ошибки, а действует осознанно и контролируемо.

По сути, это «GPS для мышления»: система не решает за человека и не лечит его, а помогает сориентироваться и принять решение в моменте. Масштабируется не за счёт личности или опыта конкретного консультанта, а за счёт формализации повторяющихся человеческих ситуаций и паттернов поведения.

## Запуск

### Требования
- Python 3.10+
- Установленные зависимости из `requirements.txt`

### Установка
1. Клонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv .venv`
3. Активируйте окружение: `source .venv/bin/activate`
4. Установите зависимости: `pip install -r requirements.txt`
5. Скопируйте `.env.example` в `.env` и заполните переменные:
   - `TG_BOT_TOKEN`: токен бота от @BotFather
   - `GROQ_API_KEY`: API ключ от Groq
6. Запустите бота: `python main.py`

### Деплой на сервер
Для продакшена используйте systemd + venv. Подробные инструкции см. в документации по деплою.
